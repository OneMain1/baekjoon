## Python Concept 관리

concept.md 파일에 Python 개념을 체계적으로 관리합니다.

### 실행 단계

1. **기존 개념 확인**: concept.md에서 유사 항목 검색
2. **중복 평가**:
   - 새 개념이면 → 적절한 섹션에 추가
   - 유사 개념이면 → 기존 항목 보강
   - 중복이면 → 추가 생략
3. **마크다운 형식 유지**

### 개념 작성 형식

```markdown
### 함수/개념 이름
간단한 설명
- 핵심 포인트 1
- 핵심 포인트 2
- 예시: `code_example`
```

### 주의사항

- 섹션별로 정리된 구조 유지
- 명확하고 간결한 설명
- 실용적인 코드 예시 포함
